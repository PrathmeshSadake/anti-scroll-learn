Quiz Result

Correct Answer

What does a closure capture?

✓ Variables from its outer scope

Explanation

A closure captures variables from its lexical (outer) scope. This means a function defined inside another function can access the parent function's variables even after the parent has returned.

Incorrect Answer

What is the output of [x*2 for x in range(3)]?

✗ [2, 4, 6]
Correct answer:
✓ [0, 2, 4]

Explanation

range(3) generates 0, 1, 2. Multiplying each by 2 gives [0, 2, 4]. Remember, range() starts at 0 by default.
